Metadata API

We expose three API endpoints:

  • select(varName, fieldName): /select?varName=x&fieldName=y
  • filter(*fieldNames): /filter?field1=x&field2=y&field3=z
  • search(query, fieldName): /search?query=x&fieldName=y

To download metadata for all variables as a CSV file, click Download metadata.

To browse variables online, click Browse online.

If you have any comments or ideas for how to improve the metadata or this API, click Feedback.
